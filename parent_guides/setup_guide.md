### Troubleshooting Common Issues

**"Python not found" error**
- Windows: Reinstall and check "Add to PATH"
- macOS/Linux: Open terminal, type `python3 --version`

**Turtle window doesn't appear**
- Some systems need additional graphics libraries
- Try installing `tkinter`: `sudo apt-get install python3-tk` (Linux)
- Or use online Python editors as backup

**Program runs but closes immediately**
- Add `turtle.done()` at end of program
- Or add `input("Press Enter to exit...")`

## üîß Technical Support

### When Your Child Gets Stuck

**First: Encourage Problem-Solving**
- "What did you expect to happen?"
- "What actually happened?"
- "What's different?"
- "Can you explain the code to me?"

**Then: Debugging Strategies**
1. **Read error messages** together
2. **Check for typos** (case-sensitive!)
3. **Test small parts** of the code
4. **Add print statements** to see values
5. **Take a break** and come back fresh

### Common Programming Errors

| Error | Common Cause | Solution |
|-------|-------------|----------|
| `SyntaxError` | Missing `:` or `()` | Check punctuation |
| `IndentationError` | Mixed tabs/spaces | Use 4 spaces consistently |
| `NameError` | Misspelled variable | Check spelling |
| `TypeError` | Wrong data type | Check `int()` vs `str()` |
| `IndexError` | List index out of range | Check `len(list)` first |
